# Get Directions

Get directions and address information for the clinic.

## Metadata

- **Name**: get_directions
- **Version**: 1.0.0
- **Category**: clinic
- **Requires Patient Context**: false

## Parameters

### from_address

- **Type**: string
- **Required**: false
- **Description**: Optional starting address for directions

## Returns

- **Type**: object
- **Description**: Clinic address and directions including street address, city, and navigation instructions
