# Get Patient Info

Retrieve patient information from CRM by phone number.

## Metadata

- **Name**: get_patient_info
- **Version**: 1.0.0
- **Category**: patient
- **Requires Patient Context**: false

## Parameters

### phone_number

- **Type**: string
- **Required**: true
- **Description**: Patient phone number

## Returns

- **Type**: object
- **Description**: Patient information from CRM including children, appointments, and contact details
